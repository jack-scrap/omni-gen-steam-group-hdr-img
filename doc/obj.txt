Objects are variables that have properties.

These properties can be of any type, and are eumerable.

Most everything in the Python language is considered an object, can inherently has properties.

## Properties
The `.` operator is used to access these properties.

# Creating
```
obj = {
	attr: 3,
	other_attr: 3
}
```

```
obj = {}
obj.attr = 3
obj.other_attr = 'Text'
```

# Accessing
```obj.property```
